*Objetos: son colecciones de propiedades, siendo una propiedad, una asociación de una clave y un valor. Un objeto contiene muchos datos sobre una misma cosa o tema. 
*Propiedades: como mencione anteriormente, una propiedad es la asociación dentro de un objeto, de un par clave: valor o Key: value. La clave es el nombre o identificador y valor es el dato que queremos almacenar en esa clave. Las claves dentro de un objeto son únicas, pero sí, varias claves pueden tenes los mismos valores. Estos valores pueden ser de cualquier tipo de datos, incluso funciones, matrices u otros objetos.
*Métodos: se llama métodos a las funciones dentro de un objeto. Se puede obtener un valor mediante un método, el cual se puede llamar mediante la notación de punto, y paréntesis finales. 
*Bucle "for ...in": es un bucle que permite recorrer las claves de un objeto. Para ello se coloca for (variable in objeto). 
*Notación de puntos (dot notation), se utiliza para hacer referencia a una propiedad en forma literal, es decir objeto.clave. Es muy sencilla de aplicar pero no se puede ocupar en todos los casos, por ej, cuando la clave es una variable. La otra notación ocupada es la de corchetes (bracket notation), la cual coloca entre comillas (salvo que sea un número o variable) y corchetes la clave, objeto["clave"]. Esta notación se emplea con el uso de variables principalmente.
